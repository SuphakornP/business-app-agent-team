# Review Data Model

Review the data model against approved use cases, screen flows, requirements, reports, and permission rules.

Inputs:

- `docs/product/05-requirements.md`

- `docs/data/01-entity-model.md`

Check:

- Missing entity

- Migration approval needs

Return findings and the standard agent result contract.
