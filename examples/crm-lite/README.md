# CRM Lite Example

Use this example when testing a small internal CRM workflow.

## Intake Prompt

```text
We need a simple CRM for sales coordinators to track leads, follow-ups, owners, next actions, and deal status.
The current workflow is a spreadsheet plus personal reminders.
Start from discovery and clarify roles, workflow, and success metrics before design.
```

## Risk Areas

- Lead ownership transfer.
- Duplicate records.
- Follow-up reminders.
- Reporting definitions.
- Visibility by team.
