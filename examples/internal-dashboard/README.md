# Internal Dashboard Example

Use this example when testing a metrics and operations dashboard.

## Intake Prompt

```text
We need an internal dashboard for managers to monitor operational requests, SLA breaches, open blockers, team workload, and weekly trends.
The source data may come from several existing systems.
Start with discovery and do not choose charts or data models until the workflow is clear.
```

## Risk Areas

- Metric definitions.
- Data source ownership.
- Refresh cadence.
- Role-based visibility.
- Actionability versus reporting-only screens.
